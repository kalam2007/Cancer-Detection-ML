# AI-Based Cancer Detection using Machine Learning

## 📌 Project Overview
This project uses machine learning to detect cancer at an early stage using the Breast Cancer dataset.

## 🧠 Algorithm Used
- Random Forest Classifier

## 📊 Features
- Data preprocessing
- Model training
- Accuracy evaluation
- Confusion matrix

## 🚀 Tools & Technologies
- Python
- Scikit-learn
- Pandas
- Google Colab

## 📈 Results
Achieved ~95% accuracy in predicting cancer (benign/malignant)

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells
